# Legal Services Platform

A modern, professional marketing and intake website for a solo attorney's productized legal services platform.

## Features

- Modern UI with Tailwind CSS
- Responsive mobile-first design
- Service pages with clear pricing
- Contact forms with validation
- SEO ready with proper metadata

## Tech Stack

- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- Inter font family

## Getting Started

### Prerequisites

- Node.js 20 LTS or higher

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
npm start
```

## Deploy to Vercel

Push to GitHub and import in Vercel for zero-config deployment.

## Customization

- Update content in `app/*/page.tsx` files
- Modify colors in `tailwind.config.js`
- Replace placeholder text with actual content
